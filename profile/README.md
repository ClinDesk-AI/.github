# ClinDesk

A private WhatsApp receptionist for clinics, running on the doctor's own Mac.

ClinDesk answers a clinic's WhatsApp in the doctor's voice — hours, prices, directions — and only wakes the doctor when a patient actually needs them. The AI runs locally on Apple Silicon (MLX). Patient data never leaves the Mac.

- **Website**: <https://clindesk.ai>
- **macOS app**: requires Apple Silicon, 16 GB RAM, macOS 14+
- **Companion app**: iPhone, end-to-end encrypted patient alerts
- **Contact**: <emin@clindesk.ai>

Most of ClinDesk lives in private repos for now. Individual components will be open-sourced as they stabilize.
